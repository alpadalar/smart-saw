### Integration Issues Found

**P1 — Missing config section:** `camera.vision` sub-section absent from config.yaml. VisionService uses hardcoded defaults (0.5s polling, 10s recording). Not configurable without code change.

**P2 — Dead config keys:** `camera.health.broken_weight` and `camera.health.wear_weight` in config.yaml are never read by HealthCalculator. Values happen to match hardcoded constants (0.7/0.3).

### Pre-existing Bug (not v2.0)

**P0 risk (edge case):** `lifecycle.py` line 31 calls `logger.warning()` before `logger` is defined at line 33. Triggers only when PySide6 is not installed. Industrial panel PC always has PySide6, so this doesn't fire in production.
