Change the constructor signature and body. Current:
```python
def __init__(self, parent=None):
    super().__init__(parent)
    self.ui = Ui_Dialog()
```

Replace with:
```python

    super().__init__(parent)
    self.ui = Ui_Dialog()
    self.ui.setupUi(self)
    self.value = initial_value
    self._prefilled = bool(initial_value)
    self.Accepted = QDialog.Accepted
    self.setup_connections()
    self.update_label()
```

The rest of `__init__` (setModal, setWindowFlags) is unchanged.

**4. NumpadDialog.add_digit() — Clear pre-fill on first keystroke**

Replace current `add_digit()`:

With:
```python

        self.update_label()
```
