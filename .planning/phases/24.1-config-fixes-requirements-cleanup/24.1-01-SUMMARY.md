## Decisions Made

1. **HealthCalculator class constants -> instance variables:** Sinif sabitleri instance degiskenine donusturulmek yerine `__init__` parametresi yapildi. Mevcut `self.BROKEN_WEIGHT` referanslari degismeden calisir.

3. **LDC_CONFIG fixtures updated:** Test fixture'larindaki config dict'ine `health` bolumu eklendi. Bu, production config yapisiyla uyumlu ve mevcut testlerin production-realistic config ile calismasini saglar.
