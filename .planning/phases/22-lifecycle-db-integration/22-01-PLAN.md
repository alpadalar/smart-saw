    **1. Create `src/services/camera/vision_service.py`** (per D-01, D-02, D-03, D-07):

    Create VisionService as a daemon thread following the exact DetectionWorker/LDCWorker pattern:
    - `class VisionService(threading.Thread)` with `daemon=True, name="vision-service"`
    - Constructor accepts: `config: dict`, `results_store: CameraResultsStore`, `camera_service: CameraService`
    - Config extraction: `polling_interval` from config.get("vision", {}).get("polling_interval", 0.5) — defaults to 0.5s (2 Hz, per discretion)
    - Config extraction: `recording_duration` from config.get("vision", {}).get("recording_duration", 10.0) — defaults to 10s (per discretion)
    - Instance state: `_stop_event = threading.Event()`, `_prev_testere_durumu: int = 0`, `_is_recording: bool = False`, `_recording_start_time: float = 0.0`

    `run()` method (main loop):
    ```python
    while not self._stop_event.is_set():
        try:
            curr = self._results_store.get("testere_durumu", 0)

            # Detect CUTTING(3) -> non-CUTTING transition (per D-03)
            if self._prev_testere_durumu == 3 and curr != 3 and not self._is_recording:
                if self._camera_service.start_recording():
                    self._is_recording = True
                    self._recording_start_time = time.monotonic()
                    logger.info("VisionService: recording started on cutting-end transition")

            # Stop recording after duration elapsed (10s per discretion)
            if self._is_recording:
                elapsed = time.monotonic() - self._recording_start_time
                if elapsed >= self._recording_duration:
                    self._camera_service.stop_recording()
                    self._is_recording = False
                    logger.info("VisionService: recording stopped after %.1fs", elapsed)

            self._prev_testere_durumu = curr
        except Exception:
            logger.warning("VisionService polling error — continuing", exc_info=True)

        self._stop_event.wait(self._polling_interval)
    ```

    `stop()` method:
    ```python
    def stop(self):
        self._stop_event.set()
    ```

    TYPE_CHECKING imports only for CameraService and CameraResultsStore (same pattern as other workers).
